# dotfiles

Personal Linux configuration files for shell, editors, terminal, window shortcuts, and development environment.

## Highlights

- Bash environment (aliases, env vars, history, options)
- Alacritty + Starship + Tmux setup
- Git configuration
- LunarVim + IdeaVim + Android Studio settings
- MPD + rmpc configuration
- Rofi themes & Cinnamon shortcuts
- Large set of cheatsheets + code gists (in `utils/`)
