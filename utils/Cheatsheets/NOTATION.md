# Modifier keys
Ctrl = C-
Alt/Meta = M-
Shift = S-
Super/Windows key = Super-
Leader (Vim/WM specific) = <leader>

# Special keys
Use angle brackets (<>) for non-alphanumeric keys

<Tab>
<Enter>
<Esc>
<Print>
<Space>

# Combination rules
- Join modifiers with - (dash)
- Separate alternatives with /
- group related with + when order doesnâ€™t matter
    Use - for normal cases (Super-h, C-s)
    Use + only when two or more modifiers must be held together (C+M-x)

# Examples
    C-M-x = Ctrl+Alt+x
    Super-<Tab> = Super+Tab
    S-<Print> = Shift+PrintScreen
    C-x C-s = press Ctrl+x, release, then Ctrl+s (sequence)
    C+M-x = press Ctrl and Alt together, then x (combo)
